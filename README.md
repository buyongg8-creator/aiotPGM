 - 튜플은? 멀티리턴하면 리스트처럼 하나로 묶어서 출력해줌
 - 하지만 튜플은 변경불가!!!

 - 리스트,튜플은 순서가 있지만 집합은 순서가없어 요솟값(출력하면 1:안녕 처럼 묶여있는것)이 안나옴
 - 특정 값 제거하기 - remove, 특정 값 제거하기 - discard 두개중에 하나 쓰면됨
 - 
 - 딕셔너리는 키에다가 값연결한것
 -  튜플은 변경 불가능한 항목들의 모음이다.
  ()을 이용하여 공백 튜플을 만들 수 있다.
  딕셔너리는 키와 값의 쌍으로 데이터를 저장한다.
  딕셔너리는 키를 이용해 값에 접근한다.
  pop() 메소드를 사용해 딕셔너리 항목을 제거할 수 있다.
  세트는 중복을 허용하지 않는 고유한 값들의 집합이다.
  set() 함수를 사용하여 세트를 생성할 수 있다.
  add() 메소드를 사용하여 세트에 요소를 추가한다.

 - 객체 지향 프로그램이란? 변수+함수해서 class개념을 만들어 오브젝트를 만든다.
 - 객체는 캡슐화시켜서 기능을 모아둠
 - 객체지향은 함수를 묶어놓은것처럼 사용할수있어 유지보수가 쉽고 정리하기가 좋음
 - 모든 객체지향은 프라이빗으로 (변수는 차단) 설정하고 필요한건 외부에서 접근할수있는 메소드를 만들고 set ,get으로 가져올수있게하면됨
